# User Guide — Steel Decision Engine

## 1. Single Material Evaluation

1. Select a material
2. Set property priorities (1–5)
3. Define process constraints
4. Click **Run Decision Engine**

Results:
- Ranked list of feasible processes
- Explanation of scoring
- Visual bar and radar plots
- Explicit rejection reasons

---

## 2. Material Comparison Mode

Compare two materials using:
- Best process vs best process
- Manual process selection

Useful for:
- Material substitution studies
- Design trade-off evaluation
- Process sensitivity analysis

---

## 3. Understanding the Results

### Scores
- Relative, not absolute
- Only comparable within a single run

### Explanations
- Highlight dominant property contributions
- Explicitly list trade-offs and compromises

### Rejected Processes
- Removed due to hard constraints
- Not negotiable by weighting

---

## 4. Output Files

- Generated automatically
- Saved as PNG
- Suitable for reports and presentations

---

## 5. Custom Materials & Processes

Edit:
- `user_materials.csv`
- `user_processes.csv`

Restart app after editing.

---

## 6. Closing the App

- Use **Stop Application** in launcher
- Closing browser tab alone does not stop backend
